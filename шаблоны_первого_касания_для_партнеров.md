# Шаблоны первого касания для партнеров Сколковской мили 2026

## Как использовать
- Для холодной базы лучше писать коротко, без вложений в первом сообщении.
- В первом касании цель не продать пакет, а получить ответ `да, интересно` / `пришлите 2-3 строки` / `давайте созвонимся`.
- Под каждого человека добавляйте 1 персональный якорь: компания, проект, отрасль, событие, город.
- PDF лучше отправлять только после первого положительного ответа или во втором сообщении.

## Формула хорошего первого сообщения
1. Коротко представиться.
2. Показать, почему пишете именно этому человеку.
3. В 1 фразе объяснить событие и смысл партнерства.
4. Задать легкий вопрос, на который удобно ответить.

- Для очень холодных контактов.
- Для людей высокого статуса.
- Когда есть риск, что длинное сообщение проигнорируют.

## Какие персональные якоря добавлять
- «Увидел ваш проект / компанию [название]».
- «Обратил внимание на ваш опыт в [индустрия]».
- «Показалось, что вам может быть близка тема спорта / сообщества / городской среды / здоровья / HR / ESG».
- «Подумал о вас, потому что у вас сильный опыт в проектах с аудиторией / сервисом / командой / lifestyle-направлением».

## Что лучше не делать
- Не отправлять сразу длинную презентацию без разрешения.
- Не писать слишком жалобно или слишком официально.
- Не начинать с просьбы о деньгах в первом же сообщении.
- Не делать первое сообщение длиннее 700-900 знаков.
- Не писать одинаковый текст всем без персонального якоря.

## Как улучшить процесс
1. Разделить контакты на 4 приоритета:
   - A: есть прямой Telegram / email / явный бренд-fit;
   - B: есть сайт и понятная компания;
   - C: есть только общий контакт компании;
   - D: контакт сомнительный или компания ликвидирована.
2. Для A и B писать в соцсети + дублировать на email через 1-2 дня.
3. Для C писать сразу коротко и просить лучший контакт внутри компании.
4. Для D не тратить много времени, а использовать только как резерв.
5. Вести в отдельной колонке статус: `не писал`, `написал в соцсети`, `отправил email`, `ответил`, `нужен follow-up`, `неактуально`.
6. Для каждого контакта заранее подготовить одну персональную причину, почему пишете именно ему.
7. Для самых сильных контактов иметь готовые 2-3 идеи интеграции под их бизнес, а не только общий партнерский пакет.

## Быстрые идеи интеграций, которые можно предлагать в диалоге
- корпоративная команда на забеге;
- подарки / сертификаты / призы участникам и победителям;
- брендированная зона на площадке;
- сервисная интеграция в сценарий дня;
- спецмеханика для сотрудников или клиентов;
- контентная поддержка и совместный storytelling вокруг темы инклюзивного спорта.
