## Cursor Cloud specific instructions

This repository is a **document storage repository**, not a software project. It contains Russian-language office documents (`.xlsx`, `.pdf`, `.docx`) related to the Skolkovo Innovation Center, plus a minimal `README.md`.

- **No application code** exists — there is nothing to build, lint, test, or run.
- **No dependencies** — no `package.json`, `requirements.txt`, `Makefile`, `Dockerfile`, `docker-compose`, or any other dependency/build configuration.
- **No services** — there are no backend, frontend, or database services.
- **No dev environment setup** is required beyond cloning the repo.

If software is added to this repository in the future, this file should be updated with relevant development instructions.
